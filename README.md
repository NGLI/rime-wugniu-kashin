# 嘉興（五縣兩區）吳語拼音輸入方案

基於[Rime中州韻輸入法引擎](https://rime.im/)的嘉興吳語拼音輸入法。

提供五個地方的輸入法，分別爲：

- 桐鄉 `wugniu_donshian`
- 海寧 `wugniu_haegnin`
- 海鹽 `wugniu_haeye`
- 嘉興 `wugniu_kashin`
- 嘉善 `wugniu_kazoe`
