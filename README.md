# 嘉興（五縣兩區）吳語拼音輸入方案

配方：℞ **NGLI/rime-wugniu_kashin**

基於 [Rime](https://rime.im/) 的嘉興吳語拼音輸入方案。

含五個地方的輸入方案：

- 桐鄉
- 海寧
- 海鹽
- 嘉興
- 嘉善

拼音方案爲 [吳語學堂式拼音](https://github.com/NGLI/rime-wugniu_kashin/wiki/音系及拼音方案) ，同時兼容 [吳語協會式拼音](http://wu-chinese.com/romanization/)。
