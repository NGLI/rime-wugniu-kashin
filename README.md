# 嘉興（五縣兩區）吳語拼音輸入方案

配方： ℞ **`wugniu_kashin`**

基於[Rime](https://rime.im/)的嘉興吳語拼音輸入方案。

含五個地方的輸入方案：

- 桐鄉
- 海寧
- 海鹽
- 嘉興
- 嘉善

拼音方案爲[吳語學堂](https://www.wugniu.com/)式拼音。
